# S1：文献检索 / Deep Research

## 什么时候使用

需要系统检索和核验相关文献时。

## 使用前需要准备

- 已完成 S0，有明确的研究问题
- 准备关键词和检索策略

## 可直接复制的 Prompt

```
# Role
你是一个科研文献检索助手。请帮我系统检索和总结相关文献。

# Research Question
[在此输入研究问题]

# Keywords
- 核心关键词：[关键词1], [关键词2], [关键词3]
- 可选的检索范围：[期刊、会议、时间范围等]

# Instructions
请帮我完成以下任务：

1. **文献检索**：检索最近 [N] 年与该问题相关的重要文献
2. **现状总结**：总结该领域的研究现状、共识和争议
3. **最接近工作**：找出最接近的 5-10 篇已有工作，说明它们做了什么、没做什么
4. **研究空白分析**：识别尚未解决的关键科学问题，标注哪些是"已确认空白"、哪些是"待核验空白"

# Output Format
请使用表格输出文献清单：

| 标题 | 作者 | 期刊/会议 | 年份 | DOI | 核验状态 |
|---|---|---|---|---|---|

# Constraints
- **必须提供每篇文献的 DOI**
- **不编造文献**：如果不确定某篇文献是否存在，明确标注"需要核验"
- 区分已阅读确认的文献和 AI 推荐（待核验）的文献
- 列出你认为最关键、最相关的工作

# Output
输出包含：
1. 研究现状总结（200-300 字）
2. 文献表格（含核对状态列）
3. 最接近已有工作分析
4. 研究空白分析（区分已确认/待核验）
```

## 期望输出

- 文献综述和表格
- 最接近工作分析
- 研究空白判定

## 人类审查重点

- [ ] 每篇文献的 DOI 是否真实
- [ ] knowledge gap 是否真实存在
