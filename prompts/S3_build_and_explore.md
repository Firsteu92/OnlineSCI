# S3：数据准备与探索 / Build and Explore

## 什么时候使用

需要准备数据、编写代码、跑通基线时。

## 使用前需要准备

- 已完成 S2（协议已冻结）
- 了解可用数据和基线方法

## 可直接复制的 Prompt

```
# Role
你是一个科研编程助手。请帮我准备数据和跑通基线。

# Task Description
[描述需要准备的数据和需要实现的方法]

# Data Sources
[描述数据来源和访问方式]

# Instructions
请帮我完成：

1. **数据准备**：
   - 下载或加载数据的脚本
   - 数据预处理流程
   - 数据质量检查
   - 生成 data_manifest.yml

2. **基线实现**：
   - 复现公开基线方法
   - 验证基线结果

3. **探索性分析**：
   - 数据基本统计
   - 可视化初步结果
   - 发现潜在问题

# Output Format
提供可直接运行的代码，包含：
- 完整的依赖说明
- 数据路径参数化
- 注释和文档

# Constraints
- 所有实验必须标记为 exploratory
- 代码必须有详细注释
- 环境依赖必须明确列出
- 不改变已冻结的协议内容

# Output
- 数据处理代码
- data_manifest.yml
- 探索性分析结果
- 环境配置说明
```

## 期望输出

- 可运行的代码
- 数据 Manifest
- 探索性分析结果

## 人类审查重点

- [ ] 数据处理是否正确
- [ ] 基线结果是否合理

## 保存位置

`projects/pXX/data/`、`projects/pXX/code/`、`projects/pXX/analysis/`
