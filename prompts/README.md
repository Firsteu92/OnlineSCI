# Prompts — 用户 Prompt 目录

本目录包含 S0–S8 各阶段对应的可直接复制的 Agent Prompt 文件。

## 使用方式

1. 在 [AI_WORKFLOW_GUIDE.md](../AI_WORKFLOW_GUIDE.md) 中判断你当前处于哪个阶段
2. 打开对应 Prompt 文件
3. 复制"可直接复制的 Prompt"部分到任意 AI Agent
4. 将 `[占位符]` 替换为你的实际内容
5. Agent 执行完成后，按 Checklist 核验并保存结果
