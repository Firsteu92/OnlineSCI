# S7：论文润色 / Paper Polish

## 什么时候使用

初稿完成后，需要改进语言质量时。

## 使用前需要准备

- 已完成论文初稿

## 可直接复制的 Prompt

```
# Role
你是一个学术论文润色专家。请改进论文的语言质量。

# Draft
[粘贴需要润色的论文全文或章节]

# Target Journal
[目标期刊，如果有特定格式要求]

# Instructions
请从以下方面改进论文：

1. **语法与拼写**：修正所有语法和拼写错误
2. **清晰度**：简化复杂句子，提高可读性
3. **术语**：确保术语使用准确一致
4. **衔接**：改善段落和句子之间的过渡
5. **期刊风格**：符合目标期刊的写作风格
6. **减少 AI 腔**：移除以下典型 AI 用语：
   - "it is worth noting"
   - "delve into"
   - "comprehensive"
   - "a deep understanding"
   - 过度 hedge

# Constraints
- **不得改变科学含义**
- **不得增加新的结果或数据**
- **不得补造引用或文献**
- **不得改写有意的技术性表述**

# Output Format
对每处修改，用【修改说明】标注理由，例如：
- 【语法修正】...
- 【清晰度提升】...
- 【术语统一】...
```

## 期望输出

- 润色后的论文
- 修改说明标注

## 人类审查重点

- [ ] 润色是否改变了科学含义
- [ ] 是否有过度修改
