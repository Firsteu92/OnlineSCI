# Checklists — 人工审查清单

本目录包含 G0–G5 六道质量门槛对应的人工审查清单。

## 使用方式

1. 完成一个 S 阶段后，找到对应的 G 门槛清单
2. 逐项检查 Agent 的输出
3. 所有必选项通过后，该门槛才算通过
4. 将检查结果记录在项目 STATUS.md 中
