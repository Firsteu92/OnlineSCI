# G1：文献与创新核验清单

**规则**：没有完成真实、可核验的文献检索，不确认研究创新性。

## 检查项

### 文献检索完整性
- [ ] 文献检索已完成，覆盖核心方向
- [ ] 每篇文献的 DOI 已提供
- [ ] 每篇引用文献已通过 Google Scholar / Crossref 核验存在
- [ ] 无编造或张冠李戴的引用
- [ ] 引用覆盖了该方向的关键文献

### 研究现状分析
- [ ] 研究现状总结完整
- [ ] 领域共识和争议已识别
- [ ] 最接近的已有工作已列出

### 创新判断
- [ ] 研究空白已识别
- [ ] 区分了"已确认空白"和"待核验空白"
- [ ] 创新性已评估
- [ ] 致命缺陷已检查

### 决定
- [ ] 最终决定为 PROCEED 或 PROCEED_WITH_CHANGES
- [ ] 如为 HOLD 或 STOP，已记录理由

## 结论

- [ ] **通过**：文献真实可核验，创新性确认，可以进入 S2
- [ ] **不通过**：未满足条件，返回完善

*记录未通过项和待补充内容*
