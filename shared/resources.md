# Resources — 外部资源

本文档汇总 OnlineSCI 使用过程中可能涉及的通用外部资源。

## 文献检索

- [Google Scholar](https://scholar.google.com/)
- [Semantic Scholar](https://www.semanticscholar.org/)
- [PubMed](https://pubmed.ncbi.nlm.nih.gov/)（生物医学相关）
- [arXiv](https://arxiv.org/)（预印本）

## 数据存储与共享

- [Zenodo](https://zenodo.org/) — 通用研究数据存档
- [Figshare](https://figshare.com/) — 研究数据共享
- [Dryad](https://datadryad.org/) — 科学数据存储

## 开放获取

- [DOAJ](https://doaj.org/) — 开放获取期刊目录
- [Unpaywall](https://unpaywall.org/) — 合法免费论文获取
