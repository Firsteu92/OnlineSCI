# P10: 项目入口

> 此项目位为空白模板，等待填写研究内容。

## 快速开始

1. 阅读 `PROJECT.md` 了解项目模板结构
2. 填写 `project.yml` 中的基本信息
3. 按照 `AI_WORKFLOW_GUIDE.md` 从 S0 阶段开始
4. 使用 `prompts/S0_project_setup.md` 建立项目

## 项目状态

请查看 [STATUS.md](STATUS.md) 了解当前状态。

## 标准目录结构

```
idea/          — 研究想法和提案
literature/    — 文献综述和参考文献
protocols/     — 实验协议
data/          — 数据 Manifest 和校验信息
code/          — 代码
analysis/      — 分析脚本
runs/          — 实验运行记录
results/       — 结果汇总
evidence/      — 证据账本
figures/       — 图表
manuscript/    — 论文手稿
logs/          — AI 交互日志
handoff/       — 项目交接文档
```
