# Data Directory

This directory stores data manifests, checksums, and small sample files.

**Large raw data is NOT stored in this repository.** See `docs/DATA_MANAGEMENT.md` for the data policy.

## Contents

- `data_manifest.yml` — metadata of all datasets used in this project
- `checksums/` — checksum files for data integrity verification
- `samples/` — small sample files for reference (optional)
